npm 包更新总结

Alex
2018-04-19
开发者
npm

常见的包更新方法有这么几种：

  1. npm update
  2. npm-check
  3. npm-check-updates

npm update

这个方法是 npm cli 自带的方法。
在开发的时候，经常会遇到 package.json 里的一些包很久不会更新的情况，你想升级下包的版本，笨一点的方法是挨个修改版本号然后 npm install 下。很费劲，这个时候可以考虑使用 npm update

总结

  1. npm update 更新完全尊重 package.json 里的语义化版本规范的。
  2. npm-check 可以很方便的更新到包的最新版本。
  3. npm-check-updates 可以控制更新到任意版本力度的更新。但是要注意，需要运行一次安装。

开发的时候推荐 npm-check-updates 。
