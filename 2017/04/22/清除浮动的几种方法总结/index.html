清除浮动的几种方法总结

Alex
2017-04-22
CSS
CSS, float

定义

浮动的框可以向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。
由于浮动框不在文档的普通流中，所以文档的普通流中的块框表现得就像浮动框不存在一样。

— W3C

问题

因为浮动元素脱离了文档流，所以包围图片和文本的 div 不占据空间。造成父元素高度塌缩。
