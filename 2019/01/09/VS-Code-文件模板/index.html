VS Code 文件模板

Alex
2019-01-09 (Updated: 2019-04-22)
开发者
VSCode, template

曾经使用 webstorm，webstorm 创建文件的时候自带模板，比如想创建一个 html 文件，则 CMD + N 并输入 html 即可创建一个带有 html 模板的文件。在使用 vscode 之后，发现 vscode 不支持这种方式，找了很多插件发现都不尽如人意。就在刚才突然想到可以使用 snippets 来实现，只要转变一下思路（为啥之前没想到呢？）。虽然实现方法不同，但都很方便，而且 snippets 也更加灵活。

拿 html 举例，如果想创建一个 html 模板，可以在 User Snippets 下创建添加自己的模板即可

html 模板 vscode 已经帮我们支持了，当你创建好一个 html 文件之后 输入 html 即可看到提示。

html template simple myself 是我创建的，HTML template 是 vscode 帮我们创建好的。
